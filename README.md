**Installing:**

- Open Chrome
- Go to chrome://extensions
- Click the Load unpacked extension
- Select the folder for the extension to install it

**Developers:**

- Fork repository
- Clone repository
- Modify file to add or change functionality
- Test by opening chrome://extensions and pressing reload link on the extension
- Bump version number in manifest.json before git push
- Push to your repository
- Create Pull Request back to original source
